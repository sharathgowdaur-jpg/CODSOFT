# CodSoft Python Internship

Hi, this repository contains the projects I completed during my Python internship at CodSoft.

## About
I worked on basic Python projects to improve my coding skills and understand problem solving.

## Projects

1. To-Do List  
A simple program to manage daily tasks. You can add, view and delete tasks.

2. Calculator  
A basic calculator using Python. It performs addition, subtraction, multiplication and division. It also handles errors like invalid input.

3. Password Generator  
This program generates a random password based on the length given by the user. It uses letters, numbers and symbols.

4. Rock Paper Scissors Game  
A simple game where the user plays against the computer. It also keeps track of score and allows multiple rounds.

5. Contact Book  
This project stores contact details like name, phone number, email and address. You can add, search, update and delete contacts.
